Features

Host popular web applications and internet services

Powered by Nix and NixOS, Nixup gives you nearly instant deployments of Fediverse services like PeerTube, Mastodon and Lemmy. Collaboration software like HedgeDoc, Cryptpad and Nextcloud. Code repository hosting with Gitea and Forgejo. And many, many more.

Deploy to servers you control

Nixup integrates with the APIs of many popular hosting providers such as Hetzner Cloud, Akamai Linode, Vultr, Microsoft Azure and Amazon Web Services to provision servers that you control. You can even bring your own server as long as you can connect to it with SSH using public key authentication.

Deploy your own web applications and services

Nixup provides the services and tooling to enable you to use the power of the Nix package manager to deploy your own static websites using popular site generators such as Jekyll and Material for MkDocs. Or web application development frameworks such as Ruby on Rails, Express, Laravel and many more. Nixup aims to make it easy to harvest the benefits of Nix without requiring a PhD in Computer Science.

Migrate easily between hosting providers

Nixup provides a safe, automated process to migrate your services and data between different hosting providers. It leverages the power of reproducible builds that the Nix package manager provides.
