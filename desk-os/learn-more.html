Learn about deskOS

Introduction

deskOS aims to be easy to use and work out of the box. It has everything you need to do common, modern computing tasks like working with office documents, browsing the web and managing files.

Installation

Warning

Installing deskOS will wipe all your data and replace your existing operating system.
Please be careful to avoid data loss! When in doubt contact a professional for assistance.
  1. Download the ISO image by clicking the Download button on the homepage
  2. Flash the image onto a USB drive with Balena Etcher or Rufus
  3. Ensure that your computer starts using UEFI and that secure boot is disabled
  4. Start your computer with the USB drive and follow the installation instructions

Using deskOS

deskOS is made to provide a familiar desktop look and feel. Click the bottom left menu to find and launch applications. You can also open the menu by pressing the Windows key, sometimes referred to as the super key. When the applications menu is open you can type part of the name of an application to search for it and press enter to launch it.

Installing additional software

Launch the Software application from the applications menu to search and install additional programs. deskOS uses Flatpak to provide you access to a large library of popular software.

Unobtrusive system updates

Important system and security updates to deskOS are installed automatically in the background. These updates are installed alongside your existing system and will not overwrite it.

In the unlikely event that your computer no longer works in the updated environment, you can easily return to the previous environment by choosing an older entry in the menu that is shown when your computer starts up.
